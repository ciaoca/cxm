<!-- 基本结构 -->
<div class="panel_page">
  <div class="inwrap">
    <!-- Content -->
  </div>
</div>
<!-- 常用结构 -->
<div id="panel_id" class="panel_page">
  <a class="bgclose" href="javascript://" rel="call_panel" rev="panel_id"></a>
  <div class="inwrap">
    <div class="header">
      <header>
        <h1></h1>
        <div class="group onright">
          <a class="icon close" href="javascript://" rel="call_panel" rev="panel_id"></a>
        </div>
      </header>
    </div>
    <div class="panel_body">
      <!-- Content -->
    </div>
  </div>
</div>

.header + .panel_body

该结构与主界面保持一致，.header位置固定。

.bgclose

用于面板非全屏时，点击遮罩层时可以关闭面板。

实际使用时，在.inwarp中自定义任何内容。

通过去除背景、遮罩后，也可以制作出类似对话框的效果。

需要在面板中使用 Tab Navigator 或 Filter Tool，将标签放置在.header与.panel_body之间。

<div id="panel_id" class="panel_page">
  <div class="inwrap">
    <div class="header">...</div>
    <div class="tabnav">...</div>
    <div class="filter_tool">...</div>
    <div class="panel_body">...</div>
  </div>
</div>
调用方法
// 参数传递元素 ID 或者元素对象均可
WebApp.panelShow('panel_id');
WebApp.panelHide(document.getElementById('panel_id'));
WebApp.panelToggle($('#panel_id'));

页面中通过<a>标签调用面板
