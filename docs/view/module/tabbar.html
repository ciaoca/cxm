底部导航

配置内容在/static/js/init.js

// 配置示例
GLOBAL.tabBarConfig = {
  menu_1: {
    name: '主菜单名称',
    link: 'url'
  },
  menu_2: {
    name: '含有子菜单',
    sub: {
      menu_3: {
        name: '子菜单名称',
        link: 'url'
      },
      menu_4: {
        name: '子菜单名称',
        link: 'url'
      }
    }
  }
};

需要使用 TabBar 的页面，添加以下标签，页面初始化时会自动构建。

<footer id="tabbar" class="tabbar" data-target="menu_1"></footer>

参数data-target可设置为需要选中的主菜单的键名。

如需动态更改 TabBar 的内容，可通过以下操作：

var newConfig = {...};
APP.buildTabBar('tabbar', newConfig, 'newTarget');
